# My Portfolio

## Description
This is my personal portfolio website showcasing my projects, skills, and contact information. It is designed to be responsive and user-friendly.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Smooth scrolling for navigation links
- Hover effects on project items
- Dynamic loading of additional projects

## How to Run the Project
1. Clone the repository.
2. Open `index.html` in your web browser.
